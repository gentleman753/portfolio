# Yuvraj Bansal - Portfolio

This repository contains the source code for my personal portfolio website, built with React, TypeScript, and Vite.

## Running Locally

**Prerequisites:** Make sure you have [Node.js](https://nodejs.org/) installed on your machine.

1.  **Clone the repository:**
    ```bash
    git clone <your-repository-url>
    cd yuvraj-bansal-portfolio
    ```

2.  **Install dependencies:**
    ```bash
    npm install
    ```

3.  **Run the development server:**
    ```bash
    npm run dev
    ```

    Open your browser and navigate to `http://localhost:5173` (or the address shown in your terminal) to see the application running.

## Deployment

This project is ready to be deployed on [Vercel](https://vercel.com/). Simply connect your Git repository to a new Vercel project, and it will be built and deployed automatically.
